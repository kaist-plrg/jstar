# JSTAR
JavaScript IR-based Semantics Extraction Toolchain

## Environment Setting
```
export JSTAR_HOME="<<JSTAR directory path>>"
export PATH="$JSTAR_HOME/bin:$PATH"
```

## Install
```
git clone --recurse-submodules git@github.com:kaist-plrg/jstar.git
cd jstar
sbt generateModel
```
